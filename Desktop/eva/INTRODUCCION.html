INTRODUCCIÓN

HTML5 es la nueva versión del lenguaje de marcado que se usa para estructurar páginas web, que actualmente todavía sigue evolucionando. Es el sucesor de HTML4, el cual se inicio a finales de 2003 pero no fue sino hasta 2007 su desarrollo gracias a W3C quien la estandarizó.
Para más información pincha en el cuadrado negro:

Tenemos que tener en cuenta que el HTML es un lenguje y presenta una serie de caracteristicas:

  1. Compatibilidad Web
  2. Conforme con SGML
  3. Sin conocimientos de programación
  4. Facilidad de creación
  5. Difusión asegurada
  6. Marcado legible
  7. Diseño formal y conciso
  8. Extensible
  9. Orientado a objetos
  10. Datos de marcado y datos carácter

Documentos bien formados:

  1. Prólogo con declaración xml completa
  2. Un único elemento raíz
  3. Cumplir reglas sintácticas

En html5, el espacio en blanco se comrpime en uno solo y los intros no se tienen en cuenta. Si queremos que dos palabras juntas que no se realize un salto al escribir sea un &snbsp (non-breaking space)

Si queremos poner algun comentario en el html: <!-- Comentario que queramos escribir -->. Este comentario solo se ve en documento, una vez que lo visualizmos en internet no.

Los elementos <span> y <div> se usan para agrupar y estructurar un documento, y se usan menudo, junto con los atributos class e id. Ambos, son importantes a lo que se refieren las CSS.

Con la etiquet a podemos hacer referencia a culaquier pagina web y abrir cualquier recursi web, pero si le añadimos el atributo target, cambiamos el modelo de abrir las paginas web. Por defecto, cuando hacemos click en un enlace, la pagina web sikucutada substituye la pagina actual y se abre en la misma ventana. Si el atributo target lleva el valor_black la pagina solicitada se abrira en una nueva ventana.

Para empezar a crear un documento html necesitamos un editor de textos, un programa en el que puedas teclear el código de tu página. Si tienes Windows en tu ordenador puedes usar el Bloc de notas o Notepad. En mi caso, he utilizado el Bloc de Notas. Otra cosa que he necesitado has sido un navegador de Internet para poder ver como estaba quedando este manual, en mi caso he utilizado Internet Explorer y Google Chrome. Una vez finalizado, lo he subido al Git.
Una vez abierto el bloc de notas he creados varios documentos con la extensión html (así, se pueden visualizar en el navegador), el tipo será guardarlo en “todos los archivos”, y la codificación en UTF-8

Antes de comenzar a explicar este pequeño manual debemos de tener en cuenta unos conceptos básicos que te pueden ayudar a entenderlo mejor. El lenguaje html trabaja con etiquetas (tags) que tienen diferentes usos. La mayoría de las etiquetas necesitan cerrarse y modifican la información contenida dentro de ellas, para abrirla con el símbolo de menos, nombre de la etiqueta y símbolo mayor (ejemplos: “<title> ,<p> <ol>”) y para cerrarlas el símbolo de menos, contrabarra, nombre de etiqueta y símbolo de mayor (ejemplos: “</title>, </p>, </ol>”) . Aunque también encontramos etiquetas vaciás como es el caso de <br> que no necesitan cerrarse y posteriormente explicaré.
