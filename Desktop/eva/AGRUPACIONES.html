AGRUPACIONES

PARRAFO <p>

Realiza un salto de párrafo, es decir de una línea a otra. No hay que confundirlo con el <br>

TEXTO PREFORMATEADO

Hace referencia a letras monoespaciadas <pre>.

LISTAS<ul>

Diferentes formas de listas
<ul> <li> Añade punto.
<ul> <ol> Ordenar.
<dt> <dd> Para definiciones (diccionario).

  • El atributo <reversed> significa que ordena del revés.
  • Para añdirel mayusculas, minusculas... type="". Añadir ejemplo -l, -a (minusculas), -A (mayusculas), -i (números romanos minusculas), -I números romanos mayusculas.
  • Para decir en que número se quiere que se empiece se pone: <start="numero">
    Estas dos últimas opciones es mejor no utilizarlas ya que en las CSS también se pueden incluir.

LINEAS HORIZONTALES <hr>

Proporciona separaciones temáticas entre dos parrafos. Por ejemplo una obra de teatro.
Es un elemento vacio, sin etiqueta de cierre.

OTROS ELEMENTOS

Tablas <table>

Las tablas estan formadas tanto por filas (verticales) como por columnas (horizontales): <table> si se le añade el <table border> significa que tiene un borde.

Tenemos que tener en cuenta:

  1. Las <tr> son filas y dentro de esta encontramos las <td> que son las celdas, ambas son etiquetas con cierre.
    Hay que mantener siempre el formato aunque por ejemplo en un celda no haya información, la dejamos en blanco y ya está.
  2. Union:El rowspan=cifra hace referencia a las celdas colspan=cifra hace referencia a las filas. Las cifras dependen en función de lo que queremos juntar. Si queremos que el encabezdo de la tabla este en negrita se utiliza la etiqueta <th>

SALTO DE LÍNEA <br>

Esta etiqueta se utiliza para hacer saltos de línea, es decir, cambiar de parrafo. Es un elemento vacio, es decir, no necesit etiqueta de cierre y se colocn al final de la la frase/palabra queremos que cambie de línea.
Es muy útil para escribir por ejemlo poesias, direcciones y similares.
Esta etiqueta no hay que confundirla con la etiqueta <p>

Encontramos también <wbr>, es un salto de línea dependiendo de la pantalla.

HYPERENLACE

HYPERENLACE CON UNA URL

Hyperenlace donde conecta con otra url exterior (Uniform Resource Locator)

<a href=”URL”>La información que sale</a>

HYPERENLACE INTERNO

Creacion de hypenlaces en el interior del documento html. Va desde un lugar a otro.

Creas el hyperenlace aqui<elto id=eltoX> comentario que queremos poner </elto> y te lleva aqui: <a href=#eltoX> texto que quieres que se vincule</a>

HYPERENLACE GENERAL

Este típo de hyperenlace se utiliza para enlazar pagias html.

<a href="nombre del documento.html"> SELECCION</a>

OTROS ELEMENTOS

Insertar imagenes

La estructura es <img src=nombredelaimagenconlaextension.png alt=Información que queremos que se vea cuando colocamos el puntero encima de la imagen width=cifra heigh=cifra>

El width hace referencia al ancho en pixeles y el height hace referencia al alto también en pixeles, por ejemplo si queremos introducir otra medida en la cifra deberemos por ejemplo poner el simbolo del porcentaje: "%". Si queremos unas dimensiones concretas, es aconsejable introducir la imagen correctamente. Si solo se modifica el width, se modifica por defecto a escala el height.

Tipos de imagenes que podemos insertar:

  • Png
  • Gif
  • Jpg

Insertar otro tipo de documentos:

  • Documento vectoriales con la extensión svg.
  • Animaciones, también puede ser extensión: png.
  • Fotogramas con la extensión gif.
  • Pdf. Hay que tener en cuenta que solo se adjunta la primera pagina del documento pdf.
